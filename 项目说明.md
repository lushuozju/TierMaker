# 极简 Tier List 项目说明

## 📦 项目概述

这是一个黑白极简风格的动画评分表生成器，参考了 `anime-role-grid` 的设计风格，但专注于 Tier List 功能。

## 🎯 核心特性

1. **黑白极简风格** - 采用纯黑白配色，简洁大方
2. **自定义评分等级** - 支持自定义评分等级（默认 S, A, B, C, D）
3. **多行布局** - 每个评分等级可以有多行，灵活排列
4. **Bangumi API 集成** - 直接搜索 Bangumi 动画数据库
5. **拖拽排序** - 支持在同一行内拖拽调整顺序
6. **本地存储** - 数据自动保存到浏览器

## 💡 使用流程

1. **配置评分等级**（可选）
   - 点击"配置评分等级"按钮
   - 添加/修改/删除等级
   - 设置等级颜色和顺序

2. **添加动画**
   - 点击空位（显示 "+"）
   - 搜索动画名称
   - 选择要添加的动画

3. **添加新行**
   - 点击等级标签旁的 "+" 按钮
   - 在同一等级下添加新行

4. **调整顺序**
   - 拖动动画卡片调整顺序

5. **删除操作**
   - 悬停卡片，点击 "×" 删除动画
   - 点击行右侧 "×" 删除行

## 📝 数据格式

### Tier 数据结构

```typescript
{
  id: "S",           // 等级 ID
  rows: [            // 行数组
    {
      id: "S-row-0", // 行 ID
      items: [        // 动画数组
        {
          id: 12345,
          name: "动画名称",
          name_cn: "中文名称",
          image: "图片URL",

    }
  ]

```


## 🌐 API 说明

### Bangumi API

项目使用 Bangumi 公开 API，无需认证即可使用：

- **搜索端点**：`POST https://api.bgm.tv/v0/search/subjects`
- **请求格式**：JSON
- **响应格式**：JSON

### 搜索参数

```json
{
  "keyword": "动画名称",
  "filter": { "type": [2] },  // 2 = 动画
  "offset": 0,
  "limit": 20
}
```

## ⚠️ 注意事项

1. **API 限制**：Bangumi API 有频率限制，频繁请求可能被限制
2. **数据存储**：数据存储在浏览器本地，清除浏览器数据会丢失
3. **跨域问题**：开发环境可能需要配置代理
4. **图片加载**：部分图片可能因 CORS 问题无法显示
