# 使用说明

## 快速开始

### 1. 安装依赖

```bash
cd tier-list-simple
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:5173

## 功能使用

### 配置评分等级

1. 点击页面右上角的 **"配置评分等级"** 按钮
2. 在弹出的配置窗口中：
   - **添加等级**：点击"添加等级"按钮
   - **修改等级**：
     - **ID**：等级的唯一标识（如 S, A, B）
     - **标签**：显示的文本（如 S, A, B）
     - **颜色**：等级标签的背景颜色（点击颜色选择器）
   - **调整顺序**：使用 ↑ ↓ 按钮调整等级显示顺序
   - **删除等级**：点击"删除"按钮（至少保留一个等级）

### 添加动画

1. 点击任意空位（显示 "+" 的虚线框）
2. 在搜索框中输入动画名称（支持中文和日文）
3. 从搜索结果中选择要添加的动画
4. 动画会自动添加到对应位置

**提示**：
- 如果搜索不到，尝试输入完整的动画名称
- 搜索结果来自 Bangumi 数据库

### 添加新行

1. 点击评分等级标签旁的 **"+"** 按钮
2. 会在该等级下添加一个新行
3. 每个等级可以有多个行，方便分类排列

### 拖拽排序

- 拖动动画卡片可以调整顺序
- 只能在同一行内拖拽
- 空位（虚线框）不能拖拽

### 删除操作

- **删除动画**：鼠标悬停在动画卡片上，点击右上角的 **"×"** 按钮
- **删除行**：点击行右侧的 **"×"** 按钮（每个等级至少保留一行）

### 数据保存

- 所有数据自动保存到浏览器本地存储
- 刷新页面后数据不会丢失
- 清除浏览器数据会丢失所有内容

## 界面说明

### 黑白极简风格

- **背景**：白色
- **边框**：黑色 2px 实线
- **文字**：黑色
- **等级标签**：可自定义颜色的背景，白色文字

### 布局结构

```
[等级标签] [动画卡片] [动画卡片] ... [删除行按钮]
[空标签]   [动画卡片] [动画卡片] ... [删除行按钮]
```

- 第一行显示等级标签和"添加行"按钮
- 后续行显示空标签占位
- 每行右侧有删除按钮（多行时显示）

## 常见问题

### Q: 搜索不到动画怎么办？

A: 
1. 尝试输入完整的动画名称
2. 尝试使用日文名称
3. 检查网络连接
4. Bangumi API 可能有频率限制，稍后再试

### Q: 如何重置所有数据？

A: 打开浏览器开发者工具（F12），在控制台执行：
```javascript
localStorage.clear()
location.reload()
```

### Q: 可以导出图片吗？

A: 当前版本暂不支持导出功能，可以使用浏览器截图工具或扩展程序。

### Q: 支持跨行拖拽吗？

A: 当前版本仅支持同一行内的拖拽排序。跨行拖拽功能正在开发中。

## 技术说明

### 数据存储

数据存储在浏览器的 `localStorage` 中，键名为 `tier-list-data` 和 `tier-config`。

### API 使用

项目使用 Bangumi 公开 API，无需配置即可使用。API 地址：`https://api.bgm.tv/v0/search/subjects`

### 浏览器兼容性

- Chrome/Edge (推荐)
- Firefox
- Safari
- 不支持 IE
